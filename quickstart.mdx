---
title: "Quickstart"
description: "Get Simple Trade running and tracking live crypto prices in minutes."
---

This guide walks you through building and launching Simple Trade from a fresh clone. For full prerequisite details, see [Installation](/installation).

<Steps>

<Step title="Verify prerequisites">

Confirm that Node.js and CMake are available before you start:

```bash
node -v
cmake --version
```

Both commands should print a version number. If either fails, follow the [installation prerequisites](/installation#install-prerequisites) to install the missing tools.

</Step>

<Step title="Build the frontend">

From inside the `MarketUI/` directory, install dependencies and build the React app:

<CodeGroup>

```powershell Windows (PowerShell)
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
npm install --legacy-peer-deps
npm run build
```

```bash macOS / Linux (Bash)
npm install
npm run build
```

</CodeGroup>

This produces `MarketUI/dist/index.html`, which the backend window loads on launch.

</Step>

<Step title="Build the backend">

From the project root, configure and compile the C++ backend:

<CodeGroup>

```powershell Windows (PowerShell)
cmake -S . -B build/ -G "MinGW Makefiles"
cd build/
nmake
```

```bash macOS / Linux (Bash)
cmake -S . -B build/
cd build/
make
```

</CodeGroup>

<Tip>
  The first build takes longer because CMake downloads all C++ dependencies (IXWebSocket, nlohmann/json, webview, and others) from GitHub. Subsequent builds skip this step.
</Tip>

</Step>

<Step title="Launch the application">

From the `build/` directory, run the compiled binary:

<CodeGroup>

```powershell Windows (PowerShell)
./Binance_Websockets.exe
```

```bash macOS / Linux (Bash)
./Binance_Websockets
```

</CodeGroup>

</Step>

<Step title="Use the application">

When the application starts, a window titled **Simple Trade** opens at 1200×800 pixels. It connects to Binance's live data stream and begins tracking **BTC/USDT** by default.

**What you'll see on launch:**

- A candlestick chart updating in real time with 1-minute candles for BTC/USDT.
- A **Watchlist** sidebar listing all six tracked coins.

**Switching coins:**

Click any coin in the Watchlist sidebar to switch the chart view. The chart reloads with that coin's live candlestick data immediately.

<Note>
  Simple Trade connects to Binance's US WebSocket stream at `wss://stream.binance.us:9443`. Ensure your network allows outbound WebSocket connections on port 9443.
</Note>

</Step>

</Steps>
