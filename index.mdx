## What Simple Trade does

Simple Trade streams live 1-minute candlestick (Kline) data from Binance for six coins — BTC, ETH, ADA, XRP, DOT, and UNI — and renders them in an interactive chart inside a native desktop window.

## How it works

<Steps>
  <Step title="C++ backend connects to Binance">
    The `Binance_Websockets` binary opens WebSocket connections to Binance's stream API for each coin and parses incoming Kline JSON into a simplified format.
  </Step>
  <Step title="Local WebSocket server relays data">
    A local WebSocket server on `127.0.0.1:8080` forwards the cleaned market data to the frontend in real time.
  </Step>
  <Step title="React frontend renders charts">
    The `useCryptoSocket` hook listens for incoming data, builds candle objects, and feeds them to the `CandlestickChart` component for live rendering.
  </Step>
  <Step title="User switches coins from the Watchlist">
    Clicking a coin in the Watchlist updates the active coin state, clears the previous chart, and begins building a fresh candlestick series from the new data stream.
  </Step>
</Steps>
