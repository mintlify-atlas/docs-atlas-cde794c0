## Overview

`CandlestickChart` renders a live 1-minute candlestick chart using the TradingView [`lightweight-charts`](https://tradingview.github.io/lightweight-charts/) library. The chart is mounted once on component mount, updated on every new candle tick, and cleared whenever the active coin changes.

<Note>
  `lightweight-charts` requires a DOM element to attach to. `CandlestickChart` creates the chart instance on a `div` via a `useRef` container ref — `chartContainerRef`. Do not render this component outside of a browser DOM context.
</Note>

---

## Props

<ParamField path="coin" type="string" required>
  The active coin symbol (e.g., `"BTCUSDT"`). When this value changes, the chart series data is cleared so stale candles from the previous coin are not shown.
</ParamField>

<ParamField path="latestCandle" type="object" required>
  The latest candle data to render. Passed directly to `series.update()` on every change.

  <Expandable title="properties">
    <ResponseField name="time" type="number">
      Candle open timestamp in Unix seconds.
    </ResponseField>
    <ResponseField name="open" type="number">
      Opening price of the candle interval.
    </ResponseField>
    <ResponseField name="high" type="number">
      Highest price during the candle interval.
    </ResponseField>
    <ResponseField name="low" type="number">
      Lowest price during the candle interval.
    </ResponseField>
    <ResponseField name="close" type="number">
      Closing price of the candle interval.
    </ResponseField>
  </Expandable>
</ParamField>

## Lifecycle

### Chart creation (on mount)

The chart is created once inside a `useEffect` with an empty dependency array. `createChart()` attaches to the container `div` and a `CandlestickSeries` is added immediately after.

```jsx CandlestickChart.jsx
useEffect(() => {
    if (!chartContainerRef.current) return;

    const chart = LightweightCharts.createChart(chartContainerRef.current, {
        width: chartContainerRef.current.clientWidth,
        height: 400,
        layout: {
            background: {color: '#0a0a0a'},
            textColor: '#d1d4dc',
        },
        grid: {
            vertLines: {color: '#1a1a1a'},
            horzLines: {color: '#1a1a1a'},
        },
        timeScale: {
            visible: true,
            timeVisible: true,
            secondsVisible: false,
            borderColor: '#2a2a2a',
        },
        rightPriceScale: {
            autoScale: true,
            borderColor: '#2a2a2a',
            scaleMargins: {top: 0.1, bottom: 0.1},
        },
        crosshair: {
            mode: LightweightCharts.CrosshairMode.Normal,
        },
    });

    seriesRef.current = chart.addSeries(LightweightCharts.CandlestickSeries, {
        upColor: '#26a69a',
        downColor: '#ef5350',
        borderUpColor: '#26a69a',
        borderDownColor: '#ef5350',
        wickUpColor: '#26a69a',
        wickDownColor: '#ef5350',
        borderVisible: true,
    });

    chartRef.current = chart;

    const handleResize = () => {
        if (chartRef.current && chartContainerRef.current) {
            chartRef.current.applyOptions({
                width: chartContainerRef.current.clientWidth,
            });
        }
    };
    window.addEventListener('resize', handleResize);

    return () => {
        window.removeEventListener('resize', handleResize);
        if (chartRef.current) {
            chartRef.current.remove();
            chartRef.current = null;
            seriesRef.current = null;
        }
    };
}, []);
```

### Coin switching (clears data)

When the `coin` prop changes, the series data is reset to an empty array so no stale candles from the previous coin remain on screen.

```jsx CandlestickChart.jsx
useEffect(() => {
    if (!seriesRef.current) return;
    seriesRef.current.setData([]);
}, [coin]);
```

### Candle updates (on every tick)

When `latestCandle` changes, `series.update()` is called. If the candle has the same timestamp as the current open candle, it updates in place. If the timestamp is new, a new candle is appended.

```jsx CandlestickChart.jsx
useEffect(() => {
    if (!seriesRef.current || !latestCandle) return;

    try {
        seriesRef.current.update(latestCandle);
    } catch (err) {
        console.warn('[CandlestickChart] update() skipped (stale candle on coin switch):', err.message)
    }
}, [latestCandle]);
```

### Cleanup (on unmount)

When the component unmounts, the `resize` event listener is removed and `chart.remove()` is called to release the `lightweight-charts` instance and its canvas elements.

## Source

```jsx CandlestickChart.jsx
import { useEffect, useRef } from 'react';
import * as LightweightCharts from 'lightweight-charts';
import '../css/CandlestickChart.css';

const CandlestickChart = ({ coin, latestCandle}) => {
    const chartContainerRef = useRef(null);
    const chartRef = useRef(null);
    const seriesRef = useRef(null);

    // Create the chart once on mount
    useEffect(() => {
        if (!chartContainerRef.current) return;

        const chart= LightweightCharts.createChart(chartContainerRef.current, {
            width: chartContainerRef.current.clientWidth,
            height: 400,
            layout: {
                background: {color: '#0a0a0a'},
                textColor: '#d1d4dc',
            },
            grid: {
                vertLines: {color: '#1a1a1a'},
                horzLines: {color: '#1a1a1a'},
            },
            timeScale: {
                visible: true,
                timeVisible: true,
                secondsVisible: false,
                borderColor: '#2a2a2a',
            },
            rightPriceScale: {
                autoScale: true,
                borderColor: '#2a2a2a',
                scaleMargins: {top: 0.1, bottom: 0.1},
            },
            crosshair: {
                mode: LightweightCharts.CrosshairMode.Normal,
            },
        });

        seriesRef.current = chart.addSeries(LightweightCharts.CandlestickSeries, {
            upColor: '#26a69a',
            downColor: '#ef5350',
            borderUpColor: '#26a69a',
            borderDownColor: '#ef5350',
            wickUpColor: '#26a69a',
            wickDownColor: '#ef5350',
            borderVisible: true,
        });

        chartRef.current = chart;

        const handleResize = () => {
            if (chartRef.current && chartContainerRef.current) {
                chartRef.current.applyOptions({
                    width: chartContainerRef.current.clientWidth,
                });
            }
        };
        window.addEventListener('resize', handleResize);

        return () => {
            window.removeEventListener('resize', handleResize);
            if (chartRef.current) {
                chartRef.current.remove();
                chartRef.current = null;
                seriesRef.current = null;
            }
        };
    }, []);

    // Resets the chart when changing coin
    useEffect(() => {
        if (!seriesRef.current) return;
        seriesRef.current.setData([])
    }, [coin]);

    // Appends candles when they have new timestamps, updates the candle if not
    useEffect(() => {
        if (!seriesRef.current || !latestCandle) return;

        try {
            seriesRef.current.update(latestCandle);
        } catch (err) {
            console.warn('[CandlestickChart] update() skipped (stale candle on coin switch):', err.message)
        }
    }, [latestCandle]);

    return <div ref={chartContainerRef} className="ChartContainer" />;
};

export default CandlestickChart;
```
